# Kar — Merged into Bidar

`jadu-kar` no longer exists as a separate command. Session start and task management are now
one command: **`/jadu-bidar`**.

- Starting a session, seeing the live backlog table, adding/updating/reprioritizing tasks,
  and the implementation handoff (numbered plan → `go`) all live in `jadu-bidar` now.
- If a project is already resolved this session, just describe what you want (add a task,
  mark something done, show the backlog) — `jadu-bidar` picks up from wherever you are.

Run `/jadu-bidar` instead. This stub exists so old muscle memory doesn't hit a dead end.
